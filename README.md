# PaperScript

PaperScript is a PaperMC updater designed to keep a server root clean while isolating its own project files and runtime data in a visible `paperscript/` directory.

Typical layout:

```text
/anydirectory/
/anydirectory/server.properties
/anydirectory/Paper-1.21.11-130.jar
/anydirectory/paperscript.sh
/anydirectory/paperscript/
/anydirectory/paperscript/paperscript.py
/anydirectory/paperscript/downloads/
/anydirectory/paperscript/backups/
/anydirectory/paperscript/logs.log
```

The full documentation lives in [paperscript/README.md](./paperscript/README.md).
